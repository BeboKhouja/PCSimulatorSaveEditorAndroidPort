How to use the Android port

Opening a file

1. Open the app
2. Click the Open file button.
3. Select your file.
4. After a bit of waiting, the results should appear in the textbox.
If the app crashes...
Follow the section below:

Opening a larger file

1. Open the app.
2. Select the "Decrypt/Encrypt to TXT" button.
3. Select the file you want to decrypt.
4. After a bit of waiting, select where you want to put the decrypted contents.
5. Open your text editor and open that saved file.

Saving your file

1. Click the save file button.
2. Select what file you want to save to.
