How to use the Android port

Opening a file

1. Open the app
2. Click the Open file button.
3. Select your file.
4. After a bit of waiting, the results should appear in the textbox.
If the app crashes...
Follow the section below:

Opening a larger file

1. Open the app.
2. Select the "Decrypt/Encrypt to TXT" button.
3. Select the file you want to decrypt.
4. Select where you want to put the decrypted contents.
5. Open your text editor and open that saved file.

Saving your file

1. Click the save file button.
2. Select what file you want to save to.

Adding objects

Adding a banner

1. Launch the app.
2. Go into the three dot menu.
3. Select Insert Object.
4. Select the banner option.
5. Choose the picture you want to add.
6. The app should lag for a long time, but thats normal.
7. That's it!

Adding any illegal item

1. Launch the app.
2. Go into the three dot menu.
3. Select Insert Object.
4. Select from: RTX4080Ti, Banner, Cube, Projector
5. That's it!

Add a flash drive

1. Launch the app.
2. Go into the three dot menu.
3. Select Insert Object.
4. Select the Flash Drive option.
5. Set the name of the drive.
6. Set the password of the drive.
7. Choose what you want to add.
8. That's it!

Add a SSD, M.2 NVMe, HDD

1. Launch the app.
2. Go into the three dot menu.
3. Select Insert Object.
4. Select the Flash Drive option.
5. Set the name of the drive.
6. Set the password of the drive.
7. Set the size of the drive.
8. Choose what you want to add.
9. That's it!

Add a custom spawnId

1. Launch the app.
