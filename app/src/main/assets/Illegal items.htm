Illegal items

In PC Simulator, normally you can only get items through the Daily Market, Print Expert, and Store.
However, there are a few items that are normally unobtainable.

Hidden, but not illegal items

You can go to the Silk Road website to buy these stuff:

PUOIO Phone

It's just a phone. An iPhone.

Super Glue

This item makes the bonding between the 2 components infinitely strong.
The only way you can remove them is to use the Remove Component tool.

Illegal items

Projector

An actual projector and it works as a display! All you have to do is to set this as the monitor for your PC. Click the Monitor tool, select the projector, select the case.

RTX4080Ti

This was originally going to be implemented as a normal item, but it was too big to fit into computers and thus was made an illegal item.
Plus, it gets a random score at benchmarks and mining.
BTW, no Ti variant of the 4080 exists.

Pillow

Just a pillow.

Cube

Just a transparent wireframe cube.
