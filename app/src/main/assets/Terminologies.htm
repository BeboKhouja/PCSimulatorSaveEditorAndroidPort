Some terminology


Save format terminology

Global applies to every object and spawnIds in some contexts.
Specific applies to a specific spawnIds in some contexts.

Illegals refers to items that cannot be obtained through the game, and you must use save editors or hacks to do so.
Saves refer to PC Simulator save files.

Source code terminology

Namespaces (or packages) (represented by this icon ) are the root of classes.
Classes (represented by this icon ) are sort of like Java classes.
Events (represented by this icon ) call other classes when something happens.
Members (or properties) (represented by this icon ) are sort of like variables, but with properties.
Operators (represented by this icon ) are sort of like overloaded operators.
Fields (or variables) (represented by this icon ) are like variables.
Methods (or voids) (represented by this icon ) are like voids in Java.
A struct (represented by this icon ) is like a struct in C.
Private refers to anything in its class that can only be accessed by itself.
Public refers to anything that can be used by other classes.
